# Testing Harness Tool

## Introduction

### Purpose
The Testing Harness Tool is a web application created to simplify the creation, testing, and publication of reports for FHIR systems, ensuring compliance with Open HIE standards. It facilitates both automated and manual testing processes, providing comprehensive validation of applications against these standards.

### Scope
This tool facilitates the entire testing lifecycle through different user roles, each with specific responsibilities:

- **Admin**: Manages user roles and oversees the overall configuration of the tool.
- **Tester**: Performs tests on the submitted requests to determine whether the applications comply with Open HIE standards.
- **Assessee**: Submits test requests by providing the necessary inputs for validation.
- **Publisher**: Reviews the test results and publishes the final report, making it accessible to all users.
